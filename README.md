# Data Structures and Algorithms in Python

This repository contains Python implementations of fundamental Data Structures and Algorithms commonly studied in Computer Science and Engineering courses.

## Contents

### Data Structures
- Linked List
- Binary Search Tree (BST)
- AVL Tree
- Red-Black Tree
- Circular Queue
- Deque
- Expression Balancing using Stack
- Infix to Postfix Conversion
- Postfix Expression Evaluation

### Sorting Algorithms
- Quick Sort

### Graph Algorithms
- Prim's Minimum Spanning Tree
- Floyd-Warshall Algorithm
- Traveling Salesman Problem (TSP)

### Dynamic Programming & Optimization
- 0/1 Knapsack Problem

### Backtracking
- N-Queens Problem

### Compression
- Huffman Coding
